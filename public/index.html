Rails Project Creation Through Templating

Apr 7th, 2014

every ruby on rails developer has a choice of gems and plugins to use in every project, or have to choose one from a list of choice. I was thinking about something by which I can automate this step. while starting a new project I had to do those repeating tasks each time.

we can automate this part with extending rails generator through templating. rails generators have api for doig this.

there is some open source project out there dedicated for this task. we can also use those. my choice is rails app composer.

just generate your new project with this command:

rails new myapp -m https://raw.github.com/RailsApps/rails-composer/master/composer.rb
